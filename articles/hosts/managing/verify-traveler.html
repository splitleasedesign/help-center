How do you verify that travelers are who they say they are?

Split Lease uses multiple verification methods to ensure guest identity and trustworthiness. Your security and peace of mind are top priorities.

Multi-Layer Verification

Every guest must verify ID, email, phone, and payment method before booking. Background checks available for additional security.

Verification Requirements

1. Government-Issued ID

Required for all guests before booking

  • Upload photo of government ID (driver's license, passport, state ID)
  • Must be current and valid (not expired)
  • AI and manual review to verify authenticity
  • Name must match payment method
  • Photo ID compared with profile photo

2. Email Verification

  • Valid email address required
  • Confirmation link sent to email
  • Must click to verify ownership
  • Used for all communications

3. Phone Verification

  • Valid phone number required
  • SMS code sent to verify
  • Must be able to receive texts
  • Used for urgent communications
  • VOIP numbers flagged for review

4. Payment Verification

  • Valid payment method required
  • Credit/debit card or bank account
  • Small verification charge (refunded)
  • Name must match ID
  • Address verification

Background Check Option

What's Included

Optional enhanced screening:

  • Criminal record check
  • Sex offender registry
  • Eviction history
  • Identity verification
  • Terrorist watch list

How to Require

  • Enable in listing settings
  • Guest pays $25 fee
  • Results within 24-48 hours
  • You see pass/fail (not full report for privacy)
  • Guest can't book until cleared

When to Use

  • High-value property
  • Live-in hosting situation
  • Long-term guests
  • Extra peace of mind

Verification Badges

What You See on Guest Profiles

  • ✓ ID Verified - Government ID checked
  • ✓ Email Verified - Email confirmed
  • ✓ Phone Verified - Phone number confirmed
  • ✓ Payment Verified - Valid payment method
  • ✓ Background Check - Enhanced screening passed
  • ✓ Verified Host - Guest is also a host (higher trust)

Verification Levels

  • Basic: ID + email + phone + payment (minimum to book)
  • Enhanced: Basic + background check
  • Trusted: Enhanced + multiple successful bookings + great reviews

Red Flags to Watch

Profile Red Flags

  • Missing verifications (shouldn't be possible, but check)
  • Profile photo doesn't match ID photo
  • Very new account (created same day)
  • Minimal profile information
  • Stock or fake-looking photo

Communication Red Flags

  • Refuses to provide additional info
  • Asks to go off-platform
  • Story inconsistent or vague
  • Won't video chat before booking
  • Pushy or aggressive

Additional Screening You Can Do

Video Introduction

  • Request video call before accepting
  • See them in person (virtually)
  • Confirm identity matches profile
  • Get better sense of personality
  • Answer questions both ways

Social Media

  • Ask for LinkedIn profile (professional guests)
  • Google their name
  • Check if social media matches story
  • Note: Don't discriminate based on protected characteristics

References

  • Ask for previous landlord reference
  • Employer verification (for work travel)
  • Personal references

Employment Verification

  • Ask where they work
  • Verify company exists
  • Check LinkedIn or company website
  • Corporate email address is good sign

What Split Lease Checks

Automated Fraud Detection

  • Multiple accounts from same person
  • Stolen payment methods
  • Suspicious booking patterns
  • Known bad actors
  • VPN/proxy usage

Manual Review

  • High-value bookings reviewed by team
  • Suspicious accounts flagged
  • ID verification spot-checked
  • Dispute history reviewed

Ongoing Monitoring

  • Guest behavior tracked
  • Multiple complaints trigger review
  • Payment issues investigated
  • Platform abuse leads to ban

Guest Reviews and Ratings

The Best Verification

Past behavior predicts future behavior:

  • 5+ positive reviews = very reliable
  • 100% positive = excellent track record
  • Recent reviews more relevant
  • Read details, not just star rating

What to Look For in Reviews

  • "Would host again" comments
  • Clean, respectful, communicative
  • No issues with payment
  • Followed house rules
  • Left space in good condition

Red Flags in Reviews

  • Multiple negative reviews
  • Cleanliness complaints
  • Communication issues
  • Rule violations
  • Damage to property
  • "Would not host again"

Host Guarantee

  • Covers guest-caused damage
  • Applies even with verified guests
  • File claim if damage occurs
  • Requires documentation

Liability Insurance

  • Protects against guest injury claims

Trust and Safety Team

24/7 Support

  • Report suspicious behavior anytime
  • Emergency support line
  • Investigation within 24 hours
  • Action taken if needed

How to Report Concerns

  1. Go to guest's profile
  2. Click "Report User"
  3. Select issue type
  4. Provide details and evidence
  5. Trust & Safety reviews
  6. Decision within 1-3 business days

Potential Actions

  • Warning to guest
  • Additional verification required
  • Booking cancelled
  • Account suspended
  • Permanent ban from platform

Best Practices for Hosts

Screen Every Guest

  • Review profile completely
  • Check all verification badges
  • Read all reviews
  • Trust your gut feeling

Communicate Before Accepting

  • Ask questions about their situation
  • Clarify house rules
  • Gauge communication style
  • Video call if uncertain

Document Everything

  • Photos before guest arrives
  • All communication through platform
  • Note any issues immediately
  • Save evidence of problems

Trust But Verify

  • Even verified guests require judgment
  • Check-in on property periodically (with notice)
  • Maintain open communication
  • Address issues early

What If Guest Isn't Who They Claimed?

During Booking Process

  • Decline booking request
  • Report to Split Lease
  • Block user

After Booking

  • Contact Split Lease immediately
  • Do not confront guest alone
  • May void booking for fraud
  • Police involvement if necessary

Unauthorized Guest

  • If different person shows up
  • Booking violation
  • You can refuse entry
  • Report to Split Lease
